ДИОКСИД ХЛОРа ДЛЯ СТЕРИЛИЗАЦИИ СЕЛЬСКОХОЗЯЙСТВа

Диоксид хлора рекомендован ВОЗ как дезинфицирующее средство класса AI.ClO2 — безопасное и высокоэффективное дезинфицирующее средство для теплиц и пахотных земель.

Инновационное применение: Дезинфекция почвы и улучшение урожая

Диоксид хлора можно использовать для стерилизации почвы и регулирования рН почвы, быстро убивая различные патогенные бактерии и различные вирусы в почве. Это может улучшить коэффициент использования удобрений и одновременно разлагать токсичные вещества.

Раствор Диоксид хлора вводят в поливную линию для обеззараживания почвы теплиц и пашни. Это может эффективно предотвратить болезни растений, такие как бактериальное увядание, корневая гниль и т. д., без вреда для урожая.

Применение в сельском хозяйстве

  • Для удаления биопленки с ирригационных линий и накопительных баков
  • Для устранения засорения капельницы
  • Для обработки поливной воды для борьбы с болезнями
  • Уменьшить количество водорослей
